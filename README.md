# Paradox Localisation Helper

> **Made with vibe coding by Kimi K2.5** 🤖

Инструмент для работы с файлами локализации модов к играм Paradox Interactive (Stellaris, Crusader Kings, Europa Universalis, Hearts of Iron и др.; тестировалось на моде на Stellaris).

## Возможности

- **Сравнение версий** — отслеживание добавленных, удалённых и изменённых ключей между версиями мода
- **Анализ перевода** — поиск недостающих переводов с подсчётом прогресса по символам
- **Комплексный отчёт** — объединение информации о новых, изменённых и непереведённых ключах
- **Генерация YML** — автоматическое создание YML-файлов для перевода по структуре оригинала
- **Слияние переводов** — объединение устаревшего перевода с обновлением
- **Сохранение настроек** — автоматическое сохранение путей и языков в конфиге

## Быстрый старт

### Требования
- .NET 10 SDK

### Установка
```bash
git clone https://github.com/username/ParadoxLocalisationHelper.git
cd ParadoxLocalisationHelper
dotnet build
dotnet run
```

### Использование

Запустите программу без аргументов:

```bash
dotnet run
# или
ParadoxLocalisationHelper.exe
```

Откроется интерактивное меню:

```
=== Paradox Localisation Helper ===

Current Settings:
  [1] New version folder:     (not set)
  [2] Old version folder:     (not set)
  [3] Outdated translation:   (not set)
  [4] Translation update:     (not set)
  [5] Source language:        english
  [6] Target language:        russian

Actions:
  [7] Generate delta report (old vs new version) + YML files
  [8] Generate missing keys report (outdated translation vs new version)
  [9] Merge outdated translation with update
 [10] Generate translation work report (missing + modified keys)

  [0] Exit
```

## Структура проекта

```
src/
├── Models/              # Модели данных
├── Configuration/       # Управление настройками
├── Cli/                 # Консольный интерфейс
├── Parsing/             # Парсер YML-файлов
├── Storage/             # Хранилище локализаций
├── Comparison/          # Сравнение версий
├── Analysis/            # Анализ перевода
├── Reporting/           # Генерация отчётов
├── Yml/                 # Запись YML-файлов
└── Merging/             # Слияние переводов
```

## Особенности

- **Поддержка формата Paradox** — корректная обработка BOM, версий ключей (`:0`, `:1`), escape-последовательностей
- **Статистика по символам** — более точная оценка объёма работы, чем подсчёт ключей
- **Структурированные YML** — созданные файлы повторяют структуру оригинала
- **Автосохранение настроек** — пути и языки сохраняются в `config.json`
